    Ayuda

    ¿Qué hace este lector?

    Lee tus libros en PDF desde el móvil, la tablet o el ordenador y recuerda por dónde vas. Hay dos formas de usarlo:

    • Abrir un PDF de tu dispositivo: funciona al instante, sin cuentas ni ajustes. Única pega: la posición de lectura se guarda solo en ese dispositivo.
    • Conectar una nube (WebDAV): tus libros y tu posición de lectura se sincronizan entre todos tus dispositivos. Necesita una configuración previa que se explica más abajo.

    ¿Qué es WebDAV?

    Es una forma estándar de acceder por internet a los archivos guardados en un servidor, como si fuese una carpeta remota. Este lector lo usa para leer tus PDF y para guardar tu posición de lectura en tu propia nube, de modo que puedas continuar desde otro dispositivo.

    ⚠️ Importante: no vale cualquier nube

    El lector funciona dentro del navegador y, por seguridad, el navegador solo permite conectar con un servidor si ese servidor lo autoriza de forma expresa (una regla técnica llamada CORS). Esto deja fuera a casi todos los servicios comerciales:

    • Google Drive, Dropbox, OneDrive: no sirven; no ofrecen un WebDAV utilizable de esta forma.
    • Koofr, pCloud, Yandex y similares: tienen WebDAV, pero bloquean el acceso desde páginas web, y no puedes cambiarlo porque el servidor no es tuyo.
    • Nextcloud u ownCloud con el permiso activado: esta es, en la práctica, la única opción que funciona para sincronizar.

    No tengo servidor propio (lo más habitual)

    Casi nadie tiene su propio servidor, y no pasa nada. Tienes dos caminos:

    • Que alguien te dé acceso a su Nextcloud (un familiar, tu centro de estudios, tu equipo de trabajo…). Pídele tres datos: la URL de tu carpeta WebDAV, tu usuario y una contraseña de aplicación. Con eso ya sincronizas entre dispositivos, sin montar nada tú.
    • Que nadie te dé acceso: usa «Abrir PDF de este dispositivo». Se lee igual de bien; solo pierdes la sincronización automática entre aparatos.

    Tengo o administro un Nextcloud / ownCloud

    Para permitir que este lector se conecte:

    • Instala la app WebAppPassword y añade el dominio de este lector (este sitio) a los orígenes permitidos.
    • Crea una contraseña de aplicación (Ajustes → Seguridad). No uses tu contraseña principal.
    • En ⚙️ Ajustes de este lector, pon la URL de tu carpeta (por ejemplo https://tu-nube.com/remote.php/dav/files/USUARIO/Libros), tu usuario y esa contraseña.

    🤖 ¿Dudas al configurar? Pregunta a una IA

    Configurar un servidor tiene su miga, pero una inteligencia artificial (ChatGPT, Claude, Gemini…) te guía paso a paso. Copia y pega preguntas como estas:

    • «Tengo un servidor Nextcloud. ¿Cómo instalo la app WebAppPassword y permito el acceso WebDAV desde una web alojada en este sitio?»
    • «¿Cómo creo una contraseña de aplicación en Nextcloud?»
    • «¿El servicio de nube [nombre] permite acceso WebDAV desde el navegador (CORS) para una web externa?»

    Privacidad
